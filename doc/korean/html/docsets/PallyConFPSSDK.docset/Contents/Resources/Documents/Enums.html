  • RenewalInterval

    License Renewal Interval. It must be the same as the time(interval) specified in the PallyCon console. If it is longer than the specified time, a license expiration error occurs, and if it is smaller than the specified time, the license cost will increases. The value received from the PallyCon license server takes precedence over the API parameter value.

    case FiveMinute = 300    /// 300 Second
case TenMinute = 600     /// 600 Second
case FifteenMinute = 900 /// 900 Second
