AES256CBC

  • encryptString(_:password:)

    returns optional encrypted string via AES-256CBC automatically generates and puts a random IV at first 16 chars the password must be exactly 32 chars long for AES-256

  • decryptString(_:password:)

    returns optional decrypted string via AES-256CBC IV need to be at first 16 chars, password must be 32 chars long

  • generatePassword()

    returns random string (uppercase & lowercase, no spaces) of 32 characters length which can be used as SHA-256 compatbile password

  • randomText(_:justLowerCase:whitespace:)

    returns random text of a defined length. Optional bool parameter justLowerCase to just generate random lowercase text and whitespace to exclude the whitespace character

  • aesEncrypt(_:key:iv:)

    returns encrypted string, IV must be 16 chars long

  • aesDecrypt(_:key:iv:)

    returns decrypted string, IV must be 16 chars long
