HLSTracksPlaylistParser

@objc
public class HLSTracksPlaylistParser : NSObject

A parser for HLS (HTTP Live Streaming) playlist files that extracts track information.

This class provides functionality to download and parse HLS master playlists (.m3u8 files), extracting information about available video, audio, and subtitle streams. It also handles PallyCon DRM-related custom information if present in the manifest.

Use this class to:

  • Download HLS manifest files from a URL
  • Parse manifests to extract media stream information
  • Access structured information about available tracks and their properties

  • downloadAndParseManifest(from:)

    Downloads and parses an HLS manifest from the specified URL.

    This method synchronously downloads and parses the manifest to return an HLSManifest object. Network communication is handled synchronously using a semaphore internally.

    Throws

    The following errors may occur:

    • Invalid URL format
    • Network connection issues
    • Server response errors
    • Download failure or content decoding failure
    • Invalid manifest format
