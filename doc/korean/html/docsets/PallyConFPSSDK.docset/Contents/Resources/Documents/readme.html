PallyCon FPS(FairPlay Streaming) SDK for iOS Document

PallyCon FPS SDK는 iOS용 미디어 서비스 앱을 개발할 때 Apple의 FairPlay Streaming DRM을 쉽게 적용할 수 있게 해주는 제품입니다. 본 문서는 SDK에 포함된 라이브러리와 샘플 프로젝트의 사용법에 대해 설명합니다.

FPS 클라이언트와 연동되는 PallyCon 멀티 DRM 서비스에 대한 설명은 별도의 온라인 가이드로 제공됩니다. SDK 사용과 관련한 기술 문의는 헬프데스크 사이트를 이용해 주시기 바랍니다.

지원 환경

  • iOS 9.0 이상
  • ARC(Automatic Reference Counting) 적용 필요

확인 사항

  • SDK는 시뮬레이터에서 동작하지 않습니다.

  • FPS HLS콘텐츠를 테스트하려면, APPLE에서 받은 App Certification과 서버 개인키를 보내주셔야 합니다.

    • Apple에서 FPS Deployment Package를 발급 받아 FPS Certification 파일과 서버 개인키, 그리고 FPS App Secret 키를 INKA로 보내주셔야 합니다.
    • PallyCon 사이트 Admin Site에 가입하여 Site ID, Site Key를 발급 받아야 합니다.

  • 위 조건이 갖춰지지 않으면 PallyConFPSSDK Sample로 FPS HLS 콘텐츠 재생이 불가합니다.

NOTE:

  • APPLE에서 받은 키가 PallyCon 사이트에 등록되면, 제공받은 패키저에 Site ID, Site Key로 HLS 콘텐츠를 암호화해야 FPS HLS 콘텐츠 재생 테스트가 가능합니다.
  • 보다 많은 정보를 원한다면 헬프데스크로 문의 주시기 바랍니다.

프로젝트 설정

다음과 같은 과정으로 PallyCon FPS SDK를 개발 프로젝트에 추가할 수 있습니다.

  1. PallyConFPSSDK.zip 파일을 받아 압축을 풉니다.

  2. lib 폴더에서 제공된 PallyConFPSSDK.xcframework 라이브러리를 프로젝트에 추가합니다.

    • 프로젝트 TARGETS에 General 탭에 Embedded Binaries에 프레임웍을 추가합니다.
    • 프로젝트 TARGETS에 Build Settings 탭에 Framework Search Paths 프레임웍 경로를 넣습니다.
    • 프로젝트 TARGETS에 Build Settings 탭에 Always Embed Swift Standard Libraries 를 Yes로 설정합니다.

  3. Bitcode 설정

    • iOS 라이브러리는 bitcode를 지원하지 않습니다.
      • 프로젝트 TARGETS에 Build Settings 탭 Enable Bitcode를 No로 설정합니다.
    • tvOS 라이브러리는 bitcode를 지원합니다.

NOTE:

  • Sample -> Advanced 샘플은 GoogleCast.framework가 필요합니다.
  • COCOAPODS을 이용하여 GoogleCast.framework을 설치(pod intstall)하고 생성된 PallyConFPSAdvanced.xcworkspace 를 실행합니다.

퀵 스타트

PallyConFPS SDK를 이용하여 FPS HLS 콘텐츠를 스트리밍 재생하거나 다운로드하는 방법은 아래 코드와 같습니다.

콘텐츠 재생

// 1. PallyConFPS SDK 객체 생성합니다.
do {
    let pallyconSdk: PallyConFPS = try PallyConFPSSDK(siteId:"Site ID", siteKey:"Site Key", fpsLicenseDelegate:"PallyConFPSLicenseDelegate")
} catch PallyConSDKException.DatabaseProcessError(let message) {
    print("PallyConFPSSDK initilize failed.\n\(message)")
}

let urlAsset = AVURLAsset(url: "Content Url(m3u8)")

// 2. FPS 콘텐츠를 재생하기 전에 Token 정보를 설정하여 라이선스를 획득 준비를 합니다.
pallyconSdk.prepare(urlAsset: urlAsset, token: "Token String")
// or
// pallyconSdk.prepare(urlAsset: urlAsset, userId:"User ID", contentId:"Content ID", token:"Token String")


// 1. PallyConFPS SDK 객체 생성합니다.
PallyConFPSSDK* pallyConSdk = [[PallyConFPSSDK alloc] initWithSiteId:"Site ID" siteKey:"Site Key" fpsLicenseDelegate:self error:nil];

AVURLAsset *urlAsset = [[AVURLAsset alloc] initWithURL:"Content Url(m3u8)" options:nil];

// 2. FPS 콘텐츠를 재생하기 전에 Token 정보를 입력하여 라이선스 획득 준비를 합니다.
[pallyconSdk prepareWithUrlAsset:urlAsset token: "Token String"]
// or
// [pallyconSdk prepareWithUrlAsset:urlAsset userId:"User ID" contentId:"Content ID" token:"Token String"]

콘텐츠 다운로드

// 1. PallyConFPS SDK 객체 생성합니다.
do {
    let pallyconSdk: PallyConFPS = try PallyConFPSSDK(siteId:"Site ID", siteKey:"Site Key", fpsLicenseDelegate:"PallyConFPSLicenseDelegate")
} catch PallyConSDKException.DatabaseProcessError(let message) {
    print("PallyConFPSSDK initilize failed.\n\(message)")
}

// 2. FPS 콘텐츠 다운로드를 위해 DownloadTask 객체를 생성하고 토큰 정보를 입력하여 라이선스 획득 준비를 합니다.
let downloadTask = pallyconSdk.createDownloadTask(url:"Content URL", token:"Token String", downloadDelegate:"PallyConFPSDownloadDelegate")
// or
// let downloadTask = pallyconSdk.createDownloadTask(url:"Content URL", userId:"User ID", contentId:"Content ID", token:"Token String", downloadDelegate:"PallyConFPSDownloadDelegate")

// 3. 콘텐츠 다운로드 시작
// 이전에 다운로드 중단된 콘텐츠일 경우, 해당 콘텐츠를 이어서 받습니다.
downloadTask.resume()

// 4. 콘텐츠 다운로드 취소 (필요 시)
downloadTask.cancel()


// 1. PallyConFPS SDK 객체 생성합니다.
PallyConFPSSDK* pallyConSdk = [[PallyConFPSSDK alloc] initWithSiteId:"Site ID" siteKey:"Site Key" fpsLicenseDelegate:self error:nil];

// 2. DownloadTask 객체 생성합니다.
// 콘텐츠 다운로드를 위해 DownloadTask 객체를 생성해서 사용합니다.
// DownloadTask 는 반드시 PallyConFPS 객체를 사용해서 생성하여 토큰 정보를 입력합니다.
[pallyConSdk createDownloadTaskWithUrl:"Content URL" token:"Token String" downloadDelegate:"PallyConFPSDownloadDelegate"]
// or
// [pallyConSdk createDownloadTaskWithUrl:"Content URL" userId:"User ID" contentId:"Content ID" token: "Token String" downloadDelegate:"PallyConFPSDownloadDelegate"]

// 3. 콘텐츠 다운로드 시작
// 이전에 다운로드 중단된 콘텐츠일 경우, 해당 콘텐츠를 이어서 받습니다.
[downloadTask resume]

// 4. 콘텐츠 다운로드 취소 (필요 시)
[downloadTask cancel]

콘텐츠 정보가 아닌 토큰을 사용하는 경우, 토큰 정보를 설정하여 아래 메소드를 호출합니다.

*/

// 크롬캐스트 SDK 를 사용할 때 'GCKMediaInformation' 객체를 만들기 위해 필요한 CustomData 가져오기

/**
콘텐츠 정보가 아닌 토큰을 사용하는 경우, 토큰 정보를 설정하여 아래 메소드를 호출합니다.
NSData* customData = [pallyconSdk getCustomDataForChromcastWithToken:"Token String"];
*/
