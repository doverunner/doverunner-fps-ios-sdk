PallyCon FPS(FairPlay Streaming) SDK for iOS Document

PallyCon FPS SDK makes it easy to apply Apple’s FairPlay Streaming DRM when developing media service apps for iOS. This document describes how to use the libraries and sample projects included in the SDK.

A description of the PallyCon Multi DRM service that works with FPS clients is provided in a separate online guide. For technical questions about using the SDK, please visit the Helpdesk site.

Supported Environments

  • iOS 9.0 or later
  • ARC(Automatic Reference Counting) required

Note:

  • This SDK does not work on Simulator.

  • The following items are needed to test your FPS HLS contents:

    • You need to acquire FPS Deployment Package from Apple, and send us FPS Certification file, Private Key file and FPS App Secret key(ASK) which are included in the package.
    • You need to sign up on PallyCon Admin Site, and get Site ID and Site Key from the site.

  • If the above conditions are not met, PallyConFPSSDK Sample can not play FPS HLS content.

NOTE:

  • Once the keys received from APPLE are registered on the PallyCon site, you need to encrypt the HLS content with Site ID and Site Key in the provided packager.
  • If you need help, please issue a ticket on Helpdesk site.

Configuring your project

You can add PallyCon FPS SDK by following steps:

  1. Extract PallyConFPSSDK.zip file.

  2. Add the PallyConFPSSDK.xcframework library provided in the lib folder to your project.

    • Add the framework to the Embedded Binaries setting in Xcode General tab.
    • Add the framework’s path to Framework Search Paths setting in Xcode Build Settings tab.
    • On the Build Settings tab on the TARGET, set the Always Embed Swift Standard Libraries to Yes.

  3. Bitcode settings

    • iOS library does not support bitcode.
      • In the project TARGETS, set the Build Settings tab Enable Bitcode to NO.
    • The tvOS library supports bitcode.

NOTE:

  • Advanced Sample requires GoogleCast.framework.
  • Install(pod intstall) the GoogleCast.framework using COCOAPODS and run the generated PallyConfPSAdvanced.xcworkspace.

Quick Start

The following code shows how to stream or download FPS HLS content using PallyConFPS SDK.

Streaming FPS content

// 1. initialize a PallyConFPS SDK.
do {
    let pallyconSdk: PallyConFPS = try PallyConFPSSDK(siteId:"Site ID", siteKey:"Site Key", fpsLicenseDelegate:"PallyConFPSLicenseDelegate")
} catch PallyConSDKException.DatabaseProcessError(let message) {
    print("PallyConFPSSDK initilize failed.\n\(message)")
}
let urlAsset = AVURLAsset(url: "Content Url(m3u8)")

// 2. Before playing FPS content, prepare to acquire a license by setting token information.
pallyconSdk.prepare(urlAsset:urlAsset, token:"Token String")
// or
// pallyconSdk.prepare(urlAsset:urlAsset, userId:"User ID",  contentId:"ContentID", token:"Token String")


// 1. initialize a PallyConFPS SDK.
PallyConFPSSDK* pallyConSdk = [[PallyConFPSSDK alloc] initWithSiteId:"Site ID" siteKey:"Site Key" fpsLicenseDelegate:self error:nil];

AVURLAsset *urlAsset = [[AVURLAsset alloc] initWithURL:"Content Url(m3u8)" options:nil];

// 2. Before playing FPS content, prepare for license acquisition by entering token information.
[pallyconSdk prepareWithUrlAsset:urlAsset token:"Token String"]
// or
// [pallyconSdk prepareWithUrlAsset:urlAsset userId:"User ID" contentId:"Content ID" token:"Token String"]

Downloading FPS content

// 1. Create PallyConFPSSDK instance.
do {
    let pallyconSdk: PallyConFPS = try PallyConFPSSDK(siteId:"Site ID", siteKey:"Site Key", fpsLicenseDelegate:"PallyConFPSLicenseDelegate")
} catch PallyConSDKException.DatabaseProcessError(let message) {
    print("PallyConFPSSDK initilize failed.\n\(message)")
}

// 2. To download FPS content, create a DownloadTask object and enter token information to prepare for license acquisition.
let downloadTask = pallyconSdk.createDownloadTask(url:"Content URL", , token:"Token String", downloadDelegate:"PallyConFPSDownloadDelegate")
// or
// let downloadTask = pallyconSdk.createDownloadTask(url:"Content URL", userId:"User ID", contentId:"Content ID", token:"Token String", downloadDelegate:"PallyConFPSDownloadDelegate")

// 3. Start downloading content
// If the content download was canceled previously, it will be resumed.
downloadTask.resume()

// 4. Cancel download (if needed)
downloadTask.cancel()

// 1. Create PallyConFPSSDK instance.
PallyConFPSSDK* pallyConSdk = [[PallyConFPSSDK alloc] initWithSiteId:"Site ID" siteKey:"Site Key" fpsLicenseDelegate:self error:nil];

// 2. Create DownloadTask instance.
// You need to use DownloadTask instance for content download.
// DownloadTask should be created by PallyConFPS instance.
[pallyConSdk createDownloadTaskWithUrl:"Content URL" token:"Token String" downloadDelegate:"PallyConFPSDownloadDelegate"]
// or
// [pallyConSdk createDownloadTaskWithUrl:"Content URL" userId:"User ID" contentId:"Content ID" token:"Token String" downloadDelegate:"PallyConFPSDownloadDelegate"]

// 3. Start downloading content
// If the content download was canceled previously, it will be resumed.
[downloadTask resume]

// 4. Cancel download (if needed)
[downloadTask cancel]

Acquiring a Token information

// Acquire a Token information
let tokenInfo: PallyConTokenInfo = PallyConFPSSDK.getTokenInfo(from:"Token String")


// Acquire a Token information
PallyConTokenInfo* tokenInfo = [PallyConFPSSDK getTokenInfoFrom:"Token String" error: nil];

Acquiring a CustomData for play the chromcast

// 'PallyCon FPS SDK' supported chromcast.
// Get CustomData needed to create 'GCKMediaInformation' object when using Chrome Cast SDK
// CustomData should be created by PallyConFPS instance.
let customData = pallyconSdk.getCustomDataForChromcast(userId: "User ID", contentId: "Content ID", optionalId: "Optional ID")
/**
If you use a token, set the token information and call the method below.
let customData = pallyconSdk.getCustomDataForChromcast(token: "Token String")
*/


// 'PallyCon FPS SDK' supported chromcast.
// Get CustomData needed to create 'GCKMediaInformation' object when using Chrome Cast SDK
// CustomData should be created by PallyConFPS instance.
NSData* customData = [pallyconSdk getCustomDataForChromcastWithUserId:"User ID" contentId: "Content ID" optionalId: "Optional ID" error: nil];
/**
If you use a token, set the token information and call the method below.
NSData* customData = [pallyconSdk getCustomDataForChromcastWithToken:"Token String"];
*/
