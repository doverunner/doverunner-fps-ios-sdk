PallyConFPSLicenseDelegate

A protocol which can be called during license acquisition.

  • fpsLicenseDidSuccessAcquiring(contentId:)

    Called when there is an success while acquiring license.

  • fpsLicense(contentId:didFailWithError:)

    Called when there is an error while acquiring license.

  • contentKeyRequest(keyData:requestData:)

    Protocol to allow content key requests. You can directly request the content key. Implement contentKeyRequest() to request a license and return Content key(Data).

  • resourceLoaderRequest(_:)

    You can modify the requestResource of AVAssetResourceLoaderDelegate.resourceLoader() used within PallyCon SDK as needed. (e.g. add subtitles to the m3u8 content you want to play) The resourceLoaderRequest() function is called when PallyConFPSSDK.mainm3u8Scheme is applied to the m3u8 URL. Example:

    // Changed [https] URL Scheme to [PallyConFPSSDK.mainm3u8Scheme]
https://somedomain.com/playlist.m3u8 -> mainm3u8://somedomain.com/playlist.m3u8

    @param requestResource An instance of AVAssetResourceLoadingRequest that provides information about the requested resource. @result TRUE if the delegate can load the resource indicated by the AVAssetResourceLoadingRequest; otherwise FALSE.
